# MultiSensor_IoT_Network_Project
Emular múltiplos sensores Shelly (Temperatura/Umidade, Porta/Janela, Inundação, Movimento) e integrá-los em um sistema que registra seus dados, fornece visualização em tempo real e histórica, e aciona alertas básicos e avançados com base em condições predefinidas, com forte ênfase em conceitos de rede
