## Descrição do Projeto
Este projeto visa emular um sistema IoT de múltiplos sensores (Temperatura/Umidade, Porta/Janela, Inundação, Movimento) utilizando Node-RED, MQTT (Mosquitto), InfluxDB e Grafana. O foco principal é demonstrar conceitos avançados de redes, como QoS, Last Will and Testament (LWT), segurança TLS e análise de tráfego com Wireshark.

## Objetivos
* Emulação Realista e Escalável de Sensores.
* Comunicação MQTT Robusta (QoS, LWT, Tópicos Estruturados).
* Persistência de Dados de Séries Temporais (InfluxDB).
* Dashboard Profissional (Grafana).
* Alertas Inteligentes (incluindo falhas de sensor via LWT).
* Análise de Tráfego com Wireshark.
* Demonstração de Segurança (MQTT TLS vs. Não Cifrado).
